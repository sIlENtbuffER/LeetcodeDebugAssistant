# Privacy Policy for LeetCode Debug Assistant

Last updated: April 24, 2026

LeetCode Debug Assistant is a Chrome extension designed to help users debug LeetCode solutions using AI.

## 1. Data Collection

This extension may access the following data:

- **Website content**: Code, problem descriptions, and test results from LeetCode pages.
- **User-provided API keys**: Stored locally on the user's device.

The extension does **not** collect personally identifiable information such as name, email, or address.

## 2. How Data Is Used

- Extracted code and problem data are sent to selected AI providers (e.g., OpenAI, Anthropic, Google Gemini) to generate debugging suggestions.
- API keys are used only to authenticate requests to these services.

## 3. Data Storage

- API keys and settings are stored locally using Chrome storage.
- No data is stored on external servers controlled by the developer.

## 4. Data Sharing

- Data is shared only with AI service providers chosen by the user.
- No data is sold or shared with third parties for advertising or analytics.

## 5. User Control

Users have full control over:

- Whether to use the extension
- Which AI provider to use
- Their API keys (can be removed at any time)

## 6. Security

We take reasonable measures to protect user data. However, users are responsible for keeping their API keys secure.

## 7. Changes

This privacy policy may be updated. Continued use of the extension implies acceptance of changes.
